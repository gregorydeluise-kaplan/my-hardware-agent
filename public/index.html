Step 1: Generate the file

Press the Windows key + R, type `msinfo32` and press Enter. Once the System Information window opens, go to "File" > "Export..." and save the file with a `.spx` extension. Alternatively, you can search for `dxdiag` in the start menu and save all information as a `.nfo` file.

Step 2: Upload the file

Drag and drop your `.spx` or `.nfo` file into the box below, or click to browse.
